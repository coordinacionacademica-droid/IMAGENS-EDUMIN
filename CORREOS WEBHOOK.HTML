Automatización de Matrículas (IMPORTRANGE)

1. Cron Job & Script

El Apps Script se ejecuta cada minuto para leer los datos del área Comercial (IMPORTRANGE). Evita duplicados marcando filas como "ENVIADO".

2. Disparo del Payload

Se inyecta un JSON al Webhook de Make con los datos vitales del alumno: Paquete, Nombre, DNI y Correo.

3. Enrutamiento Lógico

El Router lee la variable "Paquete" y divide el flujo para aplicar la experiencia adecuada según la inversión del alumno.

4. Despliegue de Valor

STD PRM VIP

Finalizando con una alerta a Telegram marcando la fila como "ENVIADO".
